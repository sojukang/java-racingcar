# 문자열 덧셈 계산기

## 기능 요구사항

- 쉼표(,) 또는 콜론(:)을 구분자로 분리한다.
- 앞의 기본 구분자(쉼표, 콜론)외에 커스텀 구분자를 지정할 수 있다.
	- "//"과 "\n" 사이의 커스텀 구분자 추출
		- (예시 : “//;\n1;2;3” => `;`가 커스텀 구분자)
	- 문자열을 구분자 기준으로 분리
- 분리된 문자열 합하여 반환한다.
	- 빈 문자열 or null 입력시 0 반환
	- 숫자 하나 입력시 해당 숫자 반환
	- 숫자 두개 컴마(,) 구분자 입력시 합 반환
- 숫자 이외의 값 or 음수 전달하는 경우 `RuntimeException` throw
